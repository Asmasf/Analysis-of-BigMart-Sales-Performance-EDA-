# Analysis of BigMart Sales Performance (EDA)

📌 Project Overview
This project analyzes BigMart sales data to understand the impact of outlet types on sales performance and identify opportunities to improve low-performing stores.

❗ Problem Statement
BigMart operates different outlet formats, but not all generate the same sales performance. Understanding these differences can support better business strategies and profitability.

❓ Business Questions
Does outlet type affect sales?
Which outlet type performs best?
What factors drive higher sales?
How can low-performing outlets improve?

📊 Dataset
Source: Kaggle
8,523 records
12 features

Key variables: Outlet Type, Outlet Size, Item MRP, Item Visibility, and Item Outlet Sales.

📈 Key Insights
Supermarket Type 1 dominates the outlet network.
Outlet type significantly impacts sales performance.
Item MRP is the strongest sales driver.
Outlet size has minimal influence on sales.

💡 Recommendations
Modernize Grocery Stores.
Focus on higher-value products.
Prioritize outlet model improvements over store expansion.
